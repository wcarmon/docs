# Setup gtk4
```bash
sudo echo "deb http://deb.debian.org/debian experimental main" > /etc/apt/sources.list.d/gtk4-rs.list
sudo chmod 644 /etc/apt/sources.list.d/gtk4-rs.list

apt update
sudo apt install -t experimental libgtk-4-1 libgtk-4-dev
```
