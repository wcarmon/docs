# Overview
- How to run go code


# CLI
- `go run ./foo/cmd/bar/...`


# Goland
- Run the main **package** not the `main.go` file
    - Run kind: Package
    - Package path: `foo/cmd/bar` (assuming a module named `foo`, and a command named `bar`)
